# Cybersecurity Research

Personal knowledge base focused on offensive security — vulnerability research,
attack techniques, and penetration testing methodology.

All techniques are mapped to [MITRE ATT&CK](https://attack.mitre.org/).

---

## Repository Structure

| Folder | Description |
|--------|-------------|
| techniques/ | Attack techniques organized by MITRE ATT&CK phase |
| vulnerabilities/ | CVE-specific research and analysis |
| playbooks/ | Full attack chains based on HTB labs |
